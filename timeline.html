Crime Index & Safety Index

Crime Index denotes an approximate index relating crime in that country or city.

Safety Index denotes an approximate index relating safety in that country or city.

I calculated these indices using crime indices values fetched from this site : www.numbeo.com/crime .

I used this formula to calculate Crime index : 
                
        sum=0;                   
        sum += 3 * level_of_crime;
        sum += crime_increasing;
        sum += 100-safe_alone_daylight;
        sum += 100-safe_alone_night;
        sum += worried_home_broken;
        sum += worried_mugged_robbed;
        sum += worried_car_stolen;
        sum += worried_things_car_stolen;
        sum += worried_attacked;
        sum += worried_insulted;
        sum += worried_skin_ethnic_religion;
        sum += problem_drugs;
        sum += problem_property_crimes;
        sum += problem_violent_crimes;
        sum += problem_corruption_bribery;

        Crime_Index = sum / 17;
        Safety_Index = 100 - Crime_Index;
